# ns-board-practice
### 🐶 개요
기술 스택 학습을 위한 게시판 프로젝트


### 🚀 기술 스택
- JDK: `1.8`
- Database: `Oracle`
- Framework: `eGovFrame 3.10.0`
- IDE: `Eclipse 2020-12`
- SQL Mapper: `MyBatis`
- Build: `Maven`
